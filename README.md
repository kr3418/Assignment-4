#2nd Ave Subway Map

This interactive map shows the next phase of the second ave subway extension of the Q train.
<img width="649" height="562" alt="Screenshot 2026-04-30 at 11 46 46 AM" src="https://github.com/user-attachments/assets/b69f8c4a-d56d-4164-bfe1-4f5fc56349cc" />

It prompts the user to click on the stations to learn more about the number of people served along with interesting locations nearby. 
The population served was estimated based on the population finder tool provided by the NYCDCP, selecting census tracts around the stations.

Using Geojason data, I traced the 2nd ave route along with the proposed extension across 125th street.
Although the current proposal is for the line to terminate on 125th and Broadway, I included the alterntive that ends at 137th street by City College.
